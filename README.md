# RAIOZ

RAIOZ is a fast-paced 2D platformer inspired by Geometry Dash, built with GameMaker. This is my first game, all the assets in this project are free.

## Gameplay

* Control a player that can move and double jump
* Avoid spikes and obstacles
* Reach the flag at the end of each level to progress
* Complete all levels as fast as possible

## Features

* 3 progressively challenging levels
* Time tracking system (mm:ss format)
* Attempt counter
* Best time record (local session)
* Instant death and restart mechanics
* Simple menu and win screen UI

## Controls

* **A / D** — Move left / right
* **W** — Jump (double jump supported)
* **Enter** — Start / Restart
* **ESC** — Exit game

## Tech Stack

* GameMaker (GML)

## How to Play

1. Open the project in GameMaker
2. Run the game
3. Press **Enter** to start
